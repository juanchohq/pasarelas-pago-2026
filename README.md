# Informe Pasarelas de Pago Chile 2026

Landing page y documento PDF con un análisis comparativo de las principales pasarelas de pago que operan en Chile en 2026.

## Contenido del repositorio

- `index.html`: Landing page estática con el resumen ejecutivo del informe.
- `informe-pasarelas-v5.pdf`: Informe completo en PDF con más de 70 páginas de análisis.
- Activos adicionales (si los agregas más adelante): imágenes, CSS o JS.

## Qué incluye el informe

- Comparativa de comisiones, tiempos de abono y medios de pago soportados.
- Análisis de pasarelas tradicionales, fintech, soluciones A2A, wallets e internacionales.
- Enfoque en casos de uso concretos: emprendedores, pymes, e‑commerce en crecimiento, SaaS y B2B.
- Recomendaciones de combinaciones de pasarelas según ticket, volumen y modelo de negocio.
